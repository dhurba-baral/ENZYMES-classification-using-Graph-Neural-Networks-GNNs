# ENZYMES Classification Using GNN

## Model Instruction

### To run the code 

1. Navigate to the folder `code`

`cd code`

2. Install the required packages using following command. 

`pip install -r requirements.txt`

Or,

`pip3 install -r requirements.txt`

3. To train all the models

`python train_model.py`

Or,

`python3 train_model.py`
##### Note: To train specific model, comment other other and run the above command;


4. To test the model function

`python test_model.py`

Or,

`python3 test_model.py`

##### To test specific model, comment other other and run the above command.
